# PM Tracking Web App

A simple web application to track preventive maintenance items.

## Features
- Add items with details: Location, Machine, Serial Number, Owner, Document.
- Validate items (move to History page).
- Remove items from Main or History page.
- Add remarks to validated items.
- Responsive design for tablets.
- Data stored in browser local storage (works offline).

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/pm-tracking.git
   ```
2. Open `index.html` in your browser or deploy on GitHub Pages.
